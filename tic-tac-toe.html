Tic-tac-toe Game

Overview

An application built using Python by me for a Milestone Project as part of an online Python course that I took for self study. Project requirements include:

  • 2 players should be able to play the game (both sitting at the same computer)
  • The board should be printed out every time a player makes a move
  • You should be able to accept input of the player position and then place a symbol on the board

Game Output

An instance of a game play is shown. In this instance, the first player has won the game.
