Shopping Cart

Overview

A web application built using ASP.NET MVC by my project team for a Continuous Assessment Project as part of my current studies. Functionalities include (bolded ones are developed by me):

  • Login
  • List Products
  • Search Products
  • Add to Cart
  • View Cart
  • Manage Activation Codes
  • View Purchase History
  • Logout

Login

Performs proper validation of userâ€™s inputs against credentials in database.

Once logged in, user will be brought to the Gallery page.

Gallery

Shows the list of products and allows user to search for items. User can add items to cart and the number of added items will be shown beside the cart icon. User can view all added cart items by clicking on the cart icon.

User will be brought to the Cart page after clicking on the cart icon.

Cart

User can update the items in cart and check out on this page. Alternatively, user can click on "Continue Shopping" to go back to Gallery page to select other items.

User can confirm purchase of items in cart by clicking on "Check out" and will be brought to the Past Purchases page which show all past purchases.

Purchases

User can view all past purchases on this page. Details such as date of purchase, quantity bought and product activation code are shown. Each purchased product has a unique activation code. If the user has purchased multiple copies of a product in the same transaction, the multiple activation codes are displayed with a combo box.

User can choose to go back to Gallery page or log out on this page.
